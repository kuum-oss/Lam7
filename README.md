## О проекте

Lam7 — это небольшое настольное приложение на C# (Windows Forms), демонстрирующее построение интерфейса из нескольких окон и использование стандартных контейнеров компоновки.

Приложение включает:

* окно входа
* окно тестирования с вкладками и динамическими элементами интерфейса
* окно результатов в табличном виде

Репозиторий задуман как аккуратный пример организации простого WinForms-проекта и работы с контейнерами интерфейса.

---

## Возможности

* переход между несколькими формами
* TabControl с динамическим добавлением вкладок
* SplitContainer для разделения рабочей области
* FlowLayoutPanel для вертикального списка элементов
* TableLayoutPanel для отображения данных в таблице
* создание элементов интерфейса во время выполнения
* корректное управление жизненным циклом форм

## Структура приложения

**Form1**
Начальное окно с вводом логина и переходом к основной части приложения.

**Form2**
Основной интерфейс:

* вкладки
* список вопросов
* динамическое добавление вкладок
* переход к окну результатов

**Form3**
Окно результатов, реализованное через табличный контейнер.

## Запуск

Через .NET CLI:

```
dotnet build
dotnet run
```

Или запуск из Visual Studio.

---

## Примечание

Логика приложения намеренно оставлена синхронной.

Приложение не выполняет сетевых запросов, операций ввода-вывода или длительных вычислений, поэтому использование async/await здесь не дало бы практической пользы. Основной фокус проекта — демонстрация контейнеров интерфейса и структуры WinForms-приложения.
